# Copying Practice

Webサイト模写練習用リポジトリ

## 構成

```
├── beginner/       # 初級（HTML/CSSの基礎）
├── elementary/     # 初中級（レイアウト・レスポンシブ）
├── intermediate/   # 中級（JavaScript基礎）
└── advanced/       # 上級（フレームワーク・アニメーション）
```

## 各レベルの内容

### Beginner（初級）
- 001〜005: HTML/CSSの基本構造、Flexbox、簡単なレイアウト

### Elementary（初中級）
- 001〜005: ヒーローセクション、グリッドレイアウト、画像ギャラリー

### Intermediate（中級）
- 001〜006: JavaScriptによるインタラクション、スライダー、アコーディオン

### Advanced（上級）
- 001〜006: アニメーション、Swiperライブラリ、Next.js/React

## ⚠️ 利用について

このリポジトリは**学習目的専用**です。

- ✅ 個人の学習・練習
- ✅ スクール受講生の課題提出
- ❌ 商用利用
- ❌ そのままの転載・再配布

※ 使用している画像素材（Unsplash等）は学習用サンプルです。
　実案件では適切な素材を使用してください。
